Welcome to the OSDDMalaria Project GitHub Page

Open Source Drug Discovery (OSDD) is the concept of drug discovery where all data and ideas are shared in real time, and anyone may participate at any level. There will be no patents. The default licence is CC-BY-3.0. Any results are both academically and commercially exploitable by whoever wishes to do so, provided the project is cited.

Raw Data

We are currently using GitHub to host the raw data for our project. A database of all OSM compounds synthesised and/or tested to date can be found on our pages in both SDF and Excel versions.

Document Sharing

Over the coming months we will be experimenting in using GitHub to share documents and workflows to see if we can use GitHub as a project management tool
